# 環境音ファイル置き場

このフォルダに音声ファイルを配置してください。ファイル名は **チャンネル ID と一致** させること。

## 必要なファイル

| ファイル名 | チャンネル | 推奨 |
|---|---|---|
| `rain.mp3` | 雨 | ループ可能な環境音 |
| `wave.mp3` | 波 | ループ可能な環境音 |
| `wind.mp3` | 風 | ループ可能な環境音 |
| `furin.mp3` | 風鈴 | ループ or 間欠音 |
| `pad.mp3` | パッド | ループ可能な持続音 |
| `insect.mp3` | 虫の音 | ループ or 間欠音 |
| `chime.mp3` | タイマー切替チャイム | 単発（1〜2 秒）。省略時は合成音にフォールバック |

## 形式

- **MP3** を推奨（`.mp3` 固定で読み込みます）
- サンプルレート・ビットレートはお好みで（目安: 128〜192 kbps）
- 環境音 6 種は **継ぎ目が目立たないループ** を推奨

## 配置例

```
public/audio/
  rain.mp3
  wave.mp3
  wind.mp3
  furin.mp3
  pad.mp3
  insect.mp3
  chime.mp3   ← 任意
```

ファイルが無いチャンネルは再生時にエラーを握りつぶし、無音のまま動作します。
