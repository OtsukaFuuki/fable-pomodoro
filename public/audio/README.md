# BGM ファイル置き場

ユーザーが用意した MP3 をここに配置します。**音声ファイルは Git に含めません**（容量のため `.gitignore` 済み）。クローン後に各自でこのフォルダへ置いてください。

## Jazz プリセット（現在）

`jazz/` フォルダに **3 ファイル** を置くと、UI で「Jazz」を選んだときに **1 → 2 → 3 → 1 …** の順でループ再生します。

```
public/audio/jazz/
  jazz1.mp3   ← 各自が配置（リポジトリには含まれない）
  jazz2.mp3
  jazz3.mp3
```

## プリセットの追加方法

`src/lib/audio-presets.ts` にエントリを足し、同様のフォルダ構成で MP3 を置いてください。
