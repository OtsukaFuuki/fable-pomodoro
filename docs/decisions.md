# 設計判断の記録（軽量 ADR）

## 音声をユーザー提供のファイル再生にした

2026-07-06: 合成音がノイズっぽく・うるさく感じられたため、`public/audio/` の MP3 を Web Audio API でループ再生する方式に変更。

2026-07-06（追記）: 6 チャンネル環境音を廃止し、プリセット方式（Jazz = jazz1〜3.mp3 の 3 重ねループ）に整理。著作権・ライセンスはユーザーが用意する音源に従う。

## 音声を完全合成にした（旧・第1〜2走者まで）

音源ファイルを使わず、全チャンネルを Web Audio API でリアルタイム合成する。著作権・ライセンス管理が不要になり、バンドルサイズが増えず、ループの継ぎ目問題が存在せず、オフラインで完全動作するため。

## タイマーを endTime 逆算にした

「残り秒をデクリメントする」のではなく「終了時刻から逆算する」。モバイルのバックグラウンドスロットリングで setInterval は止まる/間引かれるが、Date.now() は嘘をつかないため。

## タブバーなしの単一画面

機能が 1 画面で完結するため。設定はヘッダー右の軽いモーダルのみ。

## 再生状態を復元しない

起動時の突然の音出しは体験として乱暴であり、iOS の自動再生制限とも相性が悪いため。ミキサーの音量値は復元するが、必ず停止状態で起動し、ユーザー操作で鳴らす。

## 休憩中の自動フェードダウンは将来課題

初版はタイマーと環境音を完全に独立させる。「休憩中は環境音を自動でフェードダウン」は将来拡張の余地としてここに記録するのみ。
